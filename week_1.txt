1. std::pair используется,ккогда надо хранить пару упорядоченных элекментов любых типов.
std::tuple - вариативный шаблон, в которм можно хранить сколько угодно элементов.

2. std::array используется, когда массив статический.

3. std::vector для динамических ответов

4. std::deque - очередь с двумя концами, позволяет быстро добавлять элекменты в начало и конец.

5. std::forward_list - для набора элементов, в котором можно передвигаться только в одном направлении,
то есть связь с предыдущими элементами массива не нужна

6.  адаптеры контейнеров в стандартной библиотеке: stack (стек), queue (очередь),
priority_queue (очередь с приоритетами).

7. circular buffer надо использовать для реализация буферизация

8. circular buffer нельзя добавить в стандарт так как возникнут конфликты с другими объектами 
стандартной библиотеки, для которых не предусмотрено взаимодействие с нелинеаризоваными объектами

9. типы данных для работы с многомернвми массивами: array, boost::multi_array
